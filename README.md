# Al-Bayt — Bed & Breakfast

Static website for **Al-Bayt** (צימר הבית / ألبيت), a guesthouse on the banks of Lake Ram, Masaa'da — Golan Heights.

Live at **https://albayt1.com** — hosted on Cloudflare Pages, deployed automatically from this repository.

## Structure

```
index.html        the whole site (single page)
css/style.css     all styling
images/           gallery photos, logo, icons
```

## How to update

Edit `index.html` or `css/style.css` directly on GitHub (pencil icon), commit to `main`, and Cloudflare Pages redeploys automatically within a minute.

Plain HTML/CSS — no build step, no frameworks, no trackers.
